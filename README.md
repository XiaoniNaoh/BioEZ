- “**芜杂**”
- 是啊，课本厚，课程多，大家分到每门课程上的时间少

|            高中            |       大学       |
| :----------------------: | :------------: |
|          6 门科目           |   12-14 门科目    |
|          3 年学完           |     2 周学完      |
|      每门科目一周上 5-8 次课      | 每门科目一周上 1-2 次课 |

| 教辅资料丰富贴心，思维导图、知识点罗列、例题精讲 |    没教辅，自生自灭    |
