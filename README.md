## 🚀 快速入门

迫不及待想开始？跟着这几步，轻松进入 BioEZ 的世界：

1. **在线浏览**：不想下载？直接点击项目目录中的链接，在 GitHub 上查看教程，随时随地开读！
    
2. **克隆仓库**：想本地阅读？打开终端，输入以下命令把 BioEZ 抱回家：
    
    ```bash
    git clone https://github.com/XiaoniNaoh/BioEZ.git
    ```
    
3. **本地打开**：用 Markdown 编辑器（如 Obsidian 或 Typora）打开教程文件，体验更流畅！
    
4. **挑选教程**：跳转到项目目录，找到你感兴趣的主题，开始学习吧！

## 🤝 如何贡献

想加入 BioEZ 的大家庭？我们欢迎各种贡献！无论是新教程、改进现有内容，还是修复 typos，你的参与都非常重要！快来看怎么加入：

1. **Fork 仓库**：点个 Fork，把 BioEZ 复制到你的 GitHub 账号。
    
2. **创建分支**：新建一个分支，比如 git checkout -b feature/新教程。
    
3. **动手改动**：添加、编辑或修复内容，随心发挥！
    
4. **提交更改**：用清晰的提交信息，比如 git commit -m '添加遗传学新教程'。
    
5. **推送分支**：运行 git push origin feature/新教程。
    
6. **提交 Pull Request**：告诉我们你的改动，我们会尽快审核并合并！
    

> [!NOTE]
> *小提示*：所有贡献需遵守 CC BY-SA 4.0 许可证。迫不及待想看你的创意啦！

## 🙋‍♀️ Q&A

有疑问？我们来解答！

### Q1. 如何在本地查看 Markdown 文件？

A. 用支持 Markdown 的编辑器（如 Obsidian 或 Typora）打开文件，简单又清晰！

### Q2. 可以分享或修改教程吗  

A. 当然可以！根据 CC BY-SA 4.0 许可证，只要注明出处，你可以自由分享和改编。

### Q3. 这个项目适合高中生吗 ？

A. 绝对适合！虽然内容基于大学课程，但我们用通俗语言讲解，连高中生甚至更小的朋友都能看懂！

### Q4. 怎么获取最新教程 ？

A. 在 GitHub 上“Watch”我们的仓库，或者定期回来看看更新！

### Q5. 可以用这些资料教学吗 ？

A. 当然没问题！BioEZ 非常适合课堂教学或自学👍，欢迎老师和学生使用！
