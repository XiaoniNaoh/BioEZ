# 1. 氧化反应

## 1.1 油脂氧化反应的类型

**类型**
1. 自动氧化 *（自由基反应）*
2. 酶促氧化 *（自由基反应）*
3. 光敏氧化

**中间产物**
- 氢过氧化物 *（标志物）*

## 1.2 自动氧化

> 含烯底物 + 基态氧

- 无需加热
- 无需催化剂

### STEP 1：链引发

*诱导期*

- RH （引发剂） → R• + H•
- 很慢

### STEP 2：链传递

*增殖期*

- R• + $O_2$ → ROO•
- ROO• + RH → R• + ROOH

### STEP 3：链中止

- R• + R• → $R_2$
- R• + ROO• → ROOR
- ROO• + ROO• → ROOR + R2

## 1.3 光敏氧化

> **光敏氧化**：在光照下，**光敏剂**被激发，随后通过能量转移，将基态的**三线态氧**转变为高活性的**单线态氧**。

**反应步骤**
1. **第一步：光激活**：光敏剂吸收特定波长的光子，从基态（单线态）跃迁到激发单线态。
2. **第二步：系间窜越**：激发单线态的光敏剂通过系间窜越，转变为寿命较长的激发三线态。
3. **第三步：能量转移**（关键步骤）：处于激发三线态的光敏剂与基态的三线态氧发生碰撞，将能量转移给氧分子。这个过程效率很高。
4. **第四步：产物生成**：获得能量的氧分子从低能、稳定的**三线态基态氧**转变为高能、高反应活性的**单线态氧**。同时，光敏剂通常回到基态（理论上可循环，实际中可能被消耗）

- 需要**光敏剂**：叶绿素、核黄素 etc.
- 无需氧化剂
- 速度特快

## 1.4 影响油脂氧化的因素

- 脂肪酸的组成及结构
- 氧
- 温度
- 抗氧化剂
- 水分活度 *Aw*
- 助氧剂 *（如过渡金属离子）*
- 光照和射线

> [!TIP]
> **Q. 为什么猪油的货架期比植物油短？**
> 
> A. 猪油的提取过程温度高，已经经历了**链引发**，故其货架期短

# 2. 油脂在加工贮藏过程中的其他化学变化

## 2.1 油脂的水解

- 游离脂肪酸增加 → 油脂氧化速度加快 → 加速变质
- 游离脂肪酸增加 → 发烟点降低
- 油脂风味变差

> [!TIP]
> **减少游离脂肪酸的方法**
> 
> 1. 油脂精炼
> 2. 高温处理 → 酯酶失活 → 游离脂肪酸减少

## 2.2 油脂的氧化

- 食品变质败坏的主要原因
- 油脂氧化的初级产物：氢过氧化物

## 2.3 油脂的热聚合

- 单烯键 + 共轭烯键 → 发生 Diels-Alder 反应
- 反应条件：
	- 加热
	- 隔氧

> [!NOTE]
> **评价油脂品质的指标**
> 
> - **酸价** *Av* ：中和 1 g 油脂中游离脂肪酸所需的 KOH 的毫克数
> - **过氧化值** *POV* ：1 kg 油脂中活性氧的毫克数
> - **碘价** ：每 100 g 油脂所能吸收碘的质量

# 3. 油脂的精炼

- **方法分类**：
	- 物理方法
	- 化学方法

> **油脂精炼的“四脱”** ：水化脱胶、中和脱酸、吸附脱色、蒸馏脱臭


| 序号  | 方法   | 分类       | 作用                       |
| --- | ---- | -------- | ------------------------ |
| 1   | 水化脱胶 | 物理方法     | 去除蛋白质、磷脂等杂质，防止油脂高温时起泡、冒烟 |
| 2   | 中和脱酸 | **化学方法** | 用碱去除游离脂肪酸、色素等            |
| 3   | 吸附脱色 | 物理方法     | 利用活性炭、白土等吸附力强的物质去除有色物质   |
| 4   | 蒸馏脱臭 | 物理方法     | 减压蒸馏去除异味物质               |

> [!WARNING]
> **Q. 那么，油脂精炼的代价是什么呢？**
> 
> A. 精炼过程会损失一部分脂溶维生素（A、D、E、K）

# 4. 油脂的改性

## 4.1 氢化

在一定条件下，油脂与氢气发生加成反应，使油脂中的双键得以饱和的过程

> 食品安全领域中，公众一直对**氢化植物油**的安全性颇有微词，认为其氢化过程会产生大量**反式脂肪酸**，对人体健康不利。

## 4.2 酯交换

在一定条件下，油脂分子中的脂肪酸重新分布

- 可生成起酥油和人造奶油
- 和氢化相比，可减少反式脂肪酸的产生
