#生物化学 #蛋白质  

# 1. 热处理

**温和的热处理**

- 酶失活
- 抗营养因子或蛋白质毒素变性或钝化
- 利于蛋白质的消化吸收
- 产生风味物质

**过度的热处理**

- 降低营养价值
- 美拉德反应

# 2. 低温处理

- **冷却**：4 ~ 7°C，不冻结，但会抑制微生物
- **冷冻**：< -18°C，抑制微生物，影响感官品质

# 3. 脱水

**干燥方式**
- 热风
- 真空
- 冷冻
- 喷雾
- 转鼓

# 4. 辐射

一般计量的辐射对蛋白质影响不大

> [!TIP] 因此可以采用辐照灭菌，同时保持食品的品质不改变

# 5. 碱处理

- **异构化**（L- 转变为 D-）：生物活性消失
- Lys + Cys 在**碱性**环境中发生**加成反应**，产生赖氨酸丙氨酸残基 + 羊毛硫胺酸残基

# 6. 蛋白质与其他物质的作用

- 与**脂类游离基**反应：交联聚合，导致蛋白质的营养价值降低（劣化）
	- *eg. 丙三醇*
- 与**亚硝酸盐**反应：产生有害的亚硝酸胺

# 7. 乳制品风味形成

- 也即**二甲基硫化物**的形成
