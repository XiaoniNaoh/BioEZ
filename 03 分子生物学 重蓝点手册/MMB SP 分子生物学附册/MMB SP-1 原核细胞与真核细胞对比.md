|                      | 大量重复序列                                                                | -                                                                                                                                   |

|                      | 转录产物为单顺反子 mRNA                                                        | -                                                                                                                                   |
|                      | 断裂基因，有内含子结构                                                           | -                                                                                                                                   |

|                      |                                                                       |                                                                                                                                     |

|                      | 复制起始需在前复制复合体 pre-RC 指导下进行                                             | -                                                                                                                                   |
|                      | CDK 调控 pre-RC 的形成与激活                                                  | -                                                                                                                                   |

|                      | 染色体水平调控                                                               | -                                                                                                                                   |
|                      | 复制子水平调控                                                               | -                                                                                                                                   |

|                      | 3. DNA 连接酶 Ⅰ 连接冈崎片段                                                   | 3. DNA 连接酶连接                                                                                                                        |

|                      |                                                                       |                                                                                                                                     |
